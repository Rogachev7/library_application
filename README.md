# Spring-Hibernate pet-проект
____
Проект «система контроля выдачи книг читателям», который по REST API выполняет CRUD-операции.
В качестве frontend (лежит в отдельном репозитории) использован JS + React + Bootstrap.
____
Включает в себя:

- Java 15;
- Maven для управления зависимостями;
- Spring Boot, Spring Security, Spring MVC;
- База данных MySQL;
- Hibernate для работы с БД;
- JUnit для тестирования функциональности.
____

Функциональность:

- Возможность добавлять / редактировать / удалять книги;
- Возможность добавлять / редактировать / удалять читателей;
- Валидация полей при добавлении / редактировании книг / читателей;
- Авторизация и аутентификация читателей;
- Ролевые права (администратор / пользователь) для читателей;
- Возможность добавлять / удалять записи о выдаче книг читателям;
- Поиск книг и читателей по полям сущностей.
