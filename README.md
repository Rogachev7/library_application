# Электронный учёт книг
____
CRUD-проект «система контроля выдачи книг читателям», в котором для Back-end использовал Spring + Spring Security + Hibernate + MySQL,
который по REST API выполняет CRUD-операции. В качестве Front-end (лежит в отдельном репозитории) искользовал связку JS + React + Bootstrap.
