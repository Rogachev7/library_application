# Электронный учёт книг
____
CRUD-проект «система контроля выдачи книг читателям», в котором для Back-end использовал Spring + Spring Security + Hibernate + MySQL,
который по REST API выполняет CRUD-операции. В качестве Front-end (лежит в отдельном репозитории) искользовал связку JS + React + Bootstrap.
____
На Back-end стороне реализованы:
- функции получения, добавления, изменения и удаления читателей и книга, а также получение, добавление и удаление записей о выдачи книг.
Управлять доступом пользователей к страницам сайта применяя роли;
- управление доступом клиентов к данных применяя роли;
- сортировка данных по полям сущностей.
